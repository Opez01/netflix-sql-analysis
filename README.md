# netflix-sql-analysis
SQL analysis of Netflix dataset

Netflix Data Analysis using SQL

📌 Overview

This project analyzes Netflix movies and TV shows dataset using SQL to uncover trends in content, genres, and production.

🛠 Tools Used

- SQL

📊 Analysis Performed

- Distribution of Movies vs TV Shows
- Content production by country
- Trends in content release over time

🔍 Key Insights

- Movies dominate the platform compared to TV shows
- The United States produces the highest amount of content
- Content production has increased significantly over the years

📁 Files

- netflix_data.csv → dataset
- queries.sql → SQL queries

🚀 Skills Demonstrated

- Data querying using SQL
- Data aggregation and filtering
- Analytical thinking
